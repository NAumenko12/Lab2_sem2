## 1 задание (Минимальная подстрока): Даны две строки: S и T, найдите минимальную по длине подстроку S, которая содержит все символы T (с учётом кратности).
* Пример 1: S = “ADOBECODEBANC”, T = “ABC”, Результат: “BANC”
* Пример 2: S = “a”, T = “aa”, Результат: “Не найдено”
* Пример 3: S = “aaaaaaaabb”, T = “ab”, Результат: “ab”

Реализация на языках С++, Go, Python, JavaScript - ветка zadanie1

## 2 задание (Офис): В офисе всего одна переговорная комната, но поступило N заявок на проведение встреч от разных отделов. Каждая заявка имеет время начала и время окончания. Менеджеру нужно составить расписание так, чтобы провести максимально возможное количество встреч без наложений (если одна встреча заканчивается, например, в 10:00, то следующая может начаться в 10:00). На вход поступает N строк, каждая из которых содержит два числа: start (Время начала) и end (Время окончания). Определить максимальное 7 количество встреч, которые можно провести, и вывести их номера (в порядке поступления заявок). Время переводите в формат минут после начала дня,т. е. 10:00 == 600, 10:30 == 630 и так далее.
* Пример 1: N = 4, 540 600, Результат: 3, 1 3 4

570 630
600 660
660 720
* Пример 2: N = 3, 500 700, Результат: 2, 2 3

550 600
600 650
* Пример 3: N = 3, 500 600, Результат: 3, 1 2 3

600 700
700 800
  
Реализация на языках С++, Go, Swift, Kotlin - ветка zadanie2
